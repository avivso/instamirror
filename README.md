# InstaMirror 🪞

אתר סטטי שמנתח את חשבון האינסטגרם שלך ומספר לך מי אתה שם באמת: מה אתה משדר, מה התכונות שעולות מהתוכן, ולאן אפשר לקחת את זה.

**איך זה עובד:** מטא סגרה את ה-API שנתן גישה לעוקבים ולייקים, אז הדרך היחידה לקבל את כל הנתונים היא ייצוא המידע הרשמי של אינסטגרם (Settings ← Accounts Center ← Download your information, בפורמט JSON). מעלים את ה-ZIP לאתר, והכל מנותח מקומית בדפדפן. שום קובץ לא עולה לשום שרת.

## מה מקבלים

- פרופיל: שם, ביו, תמונה
- מספרים: פוסטים, עוקבים, נעקבים, הדדיים, סטוריז, לייקים, תגובות
- מי לא עוקב בחזרה, ומי מעריץ (עוקב בלי שתעקוב בחזרה)
- מתי אתה מפרסם: לפי שנים ולפי שעות
- ההאשטגים והמילים שחוזרות אצלך, ותצוגת פיד
- ניתוח AI: מי אתה באינסטגרם, מה אתה משדר, ותוכנית פעולה של 5 צעדים לכיוון שתבחר. ארבעה מנועים לבחירה:
  - **Puter** (ברירת מחדל): חינם לגמרי, בלי שום מפתח. בשימוש הראשון קופץ חלון הרשמה קצר של Puter. כולל תמונות (עם נפילה אוטומטית לטקסט).
  - **המנוי שלך ל-Claude (Pro/Max)**: רץ מקומית דרך Claude Code. לוחצים פעמיים על `InstaMirror.command` (או `node serve.js`) ונכנסים ל-http://localhost:8788. דורש [Claude Code](https://claude.com/claude-code) מותקן ומחובר. כולל תמונות.
  - **Google Gemini**: מפתח חינמי בלי כרטיס אשראי מ-[aistudio.google.com/apikey](https://aistudio.google.com/apikey). כולל תמונות.
  - **Claude API (Anthropic)**: מפתח בתשלום. כולל תמונות.

  כל המפתחות נשמרים רק ב-localStorage של הדפדפן ונשלחים ישירות לספק.

## הרצה מקומית

```sh
node serve.js
# ואז לפתוח http://localhost:8788
```

או כל שרת סטטי אחר. יש כפתור "נסו עם חשבון לדוגמה" שטוען ייצוא מזויף מ-`fixture/sample-export.zip`.

## פריסה ל-GitHub Pages

```sh
gh repo create instamirror --public --source . --push
gh api repos/{owner}/instamirror/pages -X POST -f build_type=workflow 2>/dev/null || true
```

או פשוט: Settings ← Pages ← Deploy from branch ← main. האתר כולו סטטי, אין build.

## פרטיות

- ה-ZIP מפוענח כולו בזיכרון של הדפדפן (JSZip). אין upload.
- מפתח ה-API והכיוון שבחרת נשמרים ב-localStorage בלבד.
- בניתוח ה-AI נשלחים ל-Anthropic: כיתובים, סטטיסטיקות, ועד 8 תמונות מוקטנות (אפשר לכבות).
